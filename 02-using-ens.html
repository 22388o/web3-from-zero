02 Using the Ethereum Name Service

The Ethereum Name Service, or short ENS, can be used to abstract addresses. This way we can connect to accounts and smart contracts with a human readable string. What DNS is for IPs is ENS for blockchain addresses.

Ethers.js works transparently with ENS domains, so we can simply replace the address from the last lesson with an ENS domain that resolves to that address. In case of that address a corresponding domain is devdao.eth.
